    1. **Class structure:**
       - `public sealed class SnmpExtractorService : ISnmpExtractor`
       - Constructor takes `ILogger<SnmpExtractorService>` (injected via DI)
       - Single public method: `ExtractionResult Extract(IList<Variable> varbinds, PollDefinitionDto definition)`

    2. **Extract method logic:**
       - Build OID lookup: `definition.Oids.ToDictionary(o => o.Oid, o => o)` for O(1) matching
       - Initialize mutable dictionaries: `metrics`, `labels`, `enumMetadata`
       - For each varbind:
         a. Get OID string via `varbind.Id.ToString()`
         b. Lookup in dictionary; if not found, `_logger.LogDebug(...)` and `continue`
         c. Switch on `entry.Role`:
            - `OidRole.Metric`: call `ExtractNumericValue(varbind.Data)`. If non-null, store in metrics dict. If entry has EnumMap with entries, store in enumMetadata dict. If null (non-numeric data), `_logger.LogWarning("Non-numeric SNMP data {TypeCode} for Metric role OID {Oid}, skipping", data.TypeCode, oidString)`
            - `OidRole.Label`: call `ExtractLabelValue(varbind.Data, entry.EnumMap)`. If non-null, store in labels dict.
       - Return `new ExtractionResult { Definition = definition, Metrics = metrics, Labels = labels, EnumMapMetadata = enumMetadata }`

    3. **ExtractNumericValue(ISnmpData data) -> long?** (private method):
       - Pattern match:
         - `Integer32 i` => `i.ToInt32()` (implicit long conversion)
         - `Counter32 c32` => `(long)c32.ToUInt32()`
         - `Counter64 c64` => `(long)c64.ToUInt64()`
         - `Gauge32 g` => `(long)g.ToUInt32()`
         - `TimeTicks tt` => `(long)tt.ToUInt32()`
         - `_` => `null`

    4. **ExtractLabelValue(ISnmpData data, IReadOnlyDictionary<int, string>? enumMap) -> string?** (private method):
       - If enumMap is not null AND data is Integer32:
         - `enumMap.TryGetValue(i32.ToInt32(), out var mapped)` ? `mapped` : `i32.ToInt32().ToString()`
       - Otherwise pattern match:
         - `OctetString os` => `os.ToString()`
         - `IP ip` => `ip.ToString()`
         - `Integer32 i` => `i.ToInt32().ToString()`
         - `_` => `data.ToString()`
